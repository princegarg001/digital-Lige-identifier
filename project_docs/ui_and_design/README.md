# UI & Design

Governs the front-end aesthetics overlaying the central 3D scene.

## Documents

- **[theme.md](./theme.md)**: Defines the core CSS logic, including variables, colors, and layout philosophy.
- **[new_features.md](./new_features.md)**: Feature tracker for the control side-panels and floating window interfaces.
- **`ui_components/`**: Directory housing granular UI guidelines (e.g., Apple Liquid Glass buttons, frosted glassmorphism overlays).
