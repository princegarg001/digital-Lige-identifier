# Gemini API Integrations

This directory is the ultimate reference for interacting with the Google Gemini Multimodal Live API via WebSockets.

## Documents

- **[best_practice.md](./best_practice.md)**: The gold standard Next.js implementation using Client-to-Server Ephemeral Tokens to eliminate latency.
- **[live_api.doc.md](./live_api.doc.md)**: Official reference material and code samples for the bare-metal Live API integrations.
- **[gemin_api_guide.md](./gemin_api_guide.md) & [gemin_live_api_doc.md](./gemin_live_api_doc.md)**: Detailed breakdowns of the API capabilities (barge-in, tool-calling).
- **[gemini_content_generation_doc.md](./gemini_content_generation_doc.md) & [gemini_function_calling_doc.md](./gemini_function_calling_doc.md)**: Reference material for structuring tool JSON and executing standard REST generative calls.
- **[gemini_live_best_practices.md](./gemini_live_best_practices.md)**: Consolidated best practices for managing context windows, audio resampling, and interruptions.
- **`Ephemeral-token/`**: Reference codebase for securely requesting temporary access tokens.
